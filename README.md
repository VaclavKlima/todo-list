# Todo List

A simple Todo List application built with PHP and JavaScript.

## Features

- Add, edit, and delete tasks
- Mark tasks as complete/incomplete
- Persistent storage

## Requirements

- PHP 8.2
- Composer
- Node.js & npm

## Installation

1. Clone the repository:
2. In the project directory, run:
   ```bash
   composer install
   npm install
   vite build
   php artisan migrate
   ```
